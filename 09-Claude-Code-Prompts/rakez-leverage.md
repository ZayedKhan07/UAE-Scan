# RAKEZ Leverage Prompts

Reusable prompts for maximizing RAKEZ license capabilities and identifying opportunities.

---

## License Capability Audit

```
Audit what can be done with current RAKEZ license for [BUSINESS ACTIVITY].

Determine:
- Activities explicitly permitted
- Activities that may require clarification
- Activities that require license amendment
- Activities not permitted under any circumstance
- Gray areas and how to navigate them

Provide clear yes/no/maybe for the proposed activity.
```

---

## Fastest Revenue Path Analysis

```
Identify the fastest path to revenue using RAKEZ license for [SKILL/CAPABILITY].

Evaluate options by:
- Time to first payment
- Upfront costs required
- Regulatory friction
- Market readiness
- Scalability potential

Rank options and recommend immediate action.
```

## License Limitation Workarounds

```
Explore workarounds for RAKEZ license limitation: [SPECIFIC LIMITATION].

Investigate:
- Is this a hard constraint or interpretation?
- Partnership structures that could help
- License amendments available
- Alternative approaches to achieve same goal
- Risks of each workaround

Recommend safest viable path forward.
```

---

## Upgrade Path Evaluation

```
Evaluate upgrade paths from current RAKEZ license to enable [NEW ACTIVITY].

Compare options:
- License amendment (cost, time, process)
- Additional license (cost, time, process)
- Move to different freezone
- Move to mainland
- Partnership with licensed entity

Recommend best path based on cost, time, and strategic fit.
```

## B2B Opportunity Scan

```
Scan B2B opportunities that match RAKEZ license capabilities.

Focus on:
- Services UAE businesses need
- Problems RAKEZ consultancy license can solve
- Industries with budget for external help
- Low-competition service niches
- Quick-win vs long-term opportunities

Prioritize by ease of entry and revenue potential.
```

## Compliance Check

```
Check compliance requirements for [ACTIVITY] under RAKEZ license.

Verify:
- License activity code coverage
- Additional permits needed
- Reporting or filing requirements
- Prohibited activities to avoid
- Documentation requirements
- Renewal and ongoing compliance

Flag any red flags or action items.
```
